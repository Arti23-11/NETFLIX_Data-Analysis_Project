# 🎬 Netflix Data Analytics Project

## 📌 Overview

This project focuses on analyzing Netflix data to uncover trends, patterns, and insights related to content distribution, genres, and release strategies. It demonstrates a complete data analytics workflow—from data cleaning to analysis and reporting—using Excel and Python in Jupyter Notebook.

The project is designed to showcase practical data analysis skills relevant to real-world business scenarios.

---

## 📊 Dataset

The dataset includes information about Netflix titles such as:

* Title and type (Movie/TV Show)
* Genre and category
* Release year
* Country of origin
* Duration
* Ratings and descriptions

---

## 🛠️ Tools & Technologies

* **Excel** → Data cleaning and preprocessing
* **Python (Jupyter Notebook)** → Data analysis and visualization

  * Libraries used:

    * Pandas
    * NumPy
    * Matplotlib / Seaborn

---

## 🔄 Project Workflow

### 1. Data Cleaning (Excel)

* Removed duplicates and handled missing values
* Standardized column formats
* Cleaned inconsistent text entries (genres, country names, etc.)

### 2. Data Analysis (Python - Jupyter Notebook)

* Imported cleaned dataset using Pandas
* Performed exploratory data analysis (EDA)
* Analyzed:

  * Content distribution (Movies vs TV Shows)
  * Most popular genres
  * Year-wise content release trends
  * Country-wise content production

### 3. Visualization

* Created visualizations using Matplotlib and Seaborn:

  * Bar charts
  * Line graphs
  * Distribution plots

### 4. Reporting

* Summarized key findings in a structured report
* Highlighted trends and actionable insights

---

## 📈 Dashboard

* Visual insights are presented through charts and graphs in the Jupyter Notebook
* Key metrics and trends are clearly visualized for easy interpretation
* Notebook acts as an interactive analytical dashboard

---

## ✅ Results & Insights

* Movies dominate the platform compared to TV shows
* Significant growth in content production after 2015
* Certain genres (e.g., Adventure, Comedy) are more prevalent
* Content is heavily concentrated in a few countries

---

## ▶️ How to Run the Project

1. **Excel**

   * Open the raw dataset
   * Perform data cleaning as described

2. **Jupyter Notebook**

   * Install required libraries:

     ```bash
     pip install pandas numpy matplotlib seaborn
     ```
   * Open the notebook file:

     ```bash
     jupyter notebook
     ```
   * Run all cells to reproduce the analysis
